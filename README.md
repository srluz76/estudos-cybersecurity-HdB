# estudos-cybersecurity-HdB

1. No seu computador, no Windows, abra o Prompt de Comando (ou o Git Bash, se você estiver usando
Git para Windows).
2. Instale o Git em seu PC.
3. Navegue até o diretório em seu computador onde deseja clonar o repositório usando o comando cd
nome_do_diretorio. Utilize, por exemplo, a pasta documentos.
